# WatchApp

An Apple Watch app built with SwiftUI and watchOS.

## Requirements

- Xcode 15+
- watchOS 10+
- macOS Sonoma or later

## Getting Started

1. Clone this repository
2. Open the project in Xcode
3. Select your Watch target and a paired simulator or device
4. Build and run (`⌘R`)

## Project Structure

```
WatchApp/
├── App.swift           # App entry point
├── ContentView.swift   # Root view
└── Assets.xcassets/    # Images, colors, and app icon
```
